    Checkpoint 1

    Grid 1

    Statement

    • Create a div tag with the class of “gridOne”
    • Create two paragraph tags that respectively say “Hello World” and “My Fantastic Content”
    • This grid should have two equally sized columns
    • Use inline styling to turn the text of the second paragraph tag blue

    Result

    Hello World

    My Fantastic Content

    Grid 2

    Statement

    • Create another div tag in the same HTML file
    • Make sure that none of the grids are nested inside each other
    • Give this div the class of “gridTwo”
    • Create three divs inside of gridTwo
    • Assign them each a unique class
    • Add a paragraph tag to each of the divs with whatever text you would like
    • Each child div should have its own different background color
    • Try using red, blue, and yellow
    • This grid should have three columns
    • The third column should be twice as large as the first two columns
    • The background colors will help you identify if the sizes created by your grid-template-columns are correct

    Result

    Hello World in red

    Hello World in blue

    Hello World in yellow

    Grid 3

    Statement

    • Create another div tag in the same HTML file
    • Give this div the class of “gridThree”
    • Create two child divs inside of gridThree and give them both a unique class
    • The first child div should contain a paragraph tag and the second should contain an image of your choice
    • This grid should have two rows
    • The second row should be three times as large as the first
    • Use background colors again to help make the sizes clear
    • Give gridThree a width of 500 pixels and a height of 300 pixels
    • Make sure that you give the image a width of 500 pixels and a height of 225 pixels
    • You will also need to make sure that the image will not be distorted or too large for the row

    Result

    Hello World

    Grid 4

    Statement

    • Create another div tag in the same HTML file
    • Give this div the class of “gridFour”
    • Create two child divs and give the first a unique class name and the second an ID
    • The first div should contain a paragraph tag
    • The second should contain two paragraph tags
    • GridFour should have two equally sized rows
    • Now we will deal with a nested grid
    • Select the second child div by its ID
    • Use CSS Grid to create another grid in that div
    • There should be two grid columns of equal size
    • Give gridFour a width of 500 pixels and a background color of green

    Result

    Hello World

    Create World

    Destroy World

    Grid 5

    Statement

    • Create another grid following the same instructions as in Grid 4, with the exception of the second child div
    • Give this div the class of “gridFive”
    • The second child div should have two images instead of two paragraph tags
    • The first picture should be twice as wide as the second
    • You will also have to resize these images so that they fit, try making each 200 pixels tall and wide
    • Make sure to use object-fit: cover; again

    Result

    Hello World

    Grid 6

    Statement

    • Create another div tag in the same HTML file
    • Give this div the class of “gridSix”
    • There should be two elements inside of gridSix
    • The first should be a font-awesome icon of your choice
    • The second element should be a paragraph tag
    • Choose a google font and apply it to the paragraph tag
    • GridSix should have two grid-template-columns
    • The second column, with the paragraph, should be three times the size of the first column with the icon
    • Give gridSix a width of 500px and make the background color light gray

    Result

    Hello, Create and then Destroy World
